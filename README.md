# arduino-greenhouse-temp-normalizer

An Arduino softwares that automatically turns on the exhaust fan
if the humudity, temperature inside of a greenhouse is higher
than expected.

## Module in used
- Arduino
- LCD (16x2 I2C display)
- DHT Sensor
- Relay
- Rotary Encoder
